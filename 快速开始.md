## 运行项目

```bash

cd E:\python\desktop-ai-companion

# 2. 安装依赖

# 3. 运行
python run.py
```

- 一个透明背景的窗口，显示 🤖 emoji

## 常见问题

**Q: 窗口透明背景不生效？**
A: 确保 `WA_TranslucentBackground` 已设置，且系统支持透明窗口（Windows 10+ / macOS）

**Q: 如何替换角色图片？**
A: 修改 `app/main.py` 中的 `role_label`，用 `QLabel.setPixmap()` 加载图片

**Q: API 在哪里配置？**
A: 后续在 `app/ai/client.py` 中实现，支持多 API 切换
