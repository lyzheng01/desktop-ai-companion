# 角色系统使用说明

## 当前已实现的功能

### 1. 3 个二次元角色

| 角色 | 外观 | 性格 | 颜色 |
|------|------|------|------|
| **小樱 (Sakura)** | 粉色短发少女 | 温柔治愈 | 粉色 #FFB6C1 |
| **阿白 (Shiro)** | 白发少年 + 眼镜 | 理性知性 | 白色 #E8E8E8 |
| **奈奈 (Nana)** | 双马尾猫耳少女 | 元气活泼 | 黑色 #2A2A2A |

### 2. 动画效果

每个角色都有 2 种动画：

- **Idle 呼吸动画** (6 帧) - 轻微上下浮动，模拟呼吸
- **Blink 眨眼动画** (4 帧) - 自动眨眼（约 3 秒一次，30% 概率）

### 3. 操作方式

- **单击角色** → 打开聊天窗口
- **右键菜单** → 切换角色/隐藏/退出
- **拖动角色** → 改变位置（自动保存位置）

## 替换角色图片

如果你想用更精美的图片替换当前的占位图：

### 方法 1：手动替换

1. 保持文件名不变（如 `sakura_idle_01.png`）
2. 用你的 PNG 图片覆盖 `assets/characters/` 目录下的文件
3. 图片建议尺寸：200x300px（透明背景）

### 方法 2：使用 Live2D（高级）

后续可以集成 Live2D 模型，实现更流畅的动画：

```python
# 后续实现
from .ui.live2d_widget import Live2DWidget
# 替换 CharacterWidget 为 Live2DWidget
```

## 扩展新角色

添加新角色的步骤：

1. 在 `assets/generate_characters.py` 中添加角色配置
2. 运行脚本生成图片：`python assets/generate_characters.py`
3. 在 `app/main.py` 的右键菜单中添加切换选项

示例配置：
```python
{"name": "yuki", "color": "#E0F0FF", "label": "雪乃"},
```
