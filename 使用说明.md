# 桌面 AI 伙伴 - 使用说明

## 快速开始

### 运行项目
```bash
cd E:\python\desktop-ai-companion
python run.py
```

### 基本操作

| 操作 | 效果 |
|------|------|
| **左键点击角色** | 打开聊天窗口 |
| **拖动角色** | 移动角色位置（自动保存） |
| **右键菜单** | 切换角色/隐藏/退出 |
| **点击头部** | 惊讶表情 |
| **点击身体** | 微笑 |
| **点击手臂** | 挥手 |

## 角色说明

### 当前支持的角色

#### 1. Live2D 模型（推荐）
- **来源**: Live2D 官方免费样例 - Kei
- **位置**: `assets/live2d/kei_en/`
- **特点**: 真正的二次元风格，专业制作

- 🌸 **小樱 (Sakura)** - 粉色短发少女

1. 右键点击角色
2. 选择 "🎭 切换角色"
3. 选择想要的角色

## 动画效果

### 自动动画
- **呼吸** - 身体轻微上下浮动（1.5 秒周期）
- **眨眼** - 自动眨眼（3 秒间隔，60% 概率）

### 互动动画
- **挥手** - 点击手臂或拖动时
- **惊讶** - 鼠标移入或点击头部
- **微笑** - 点击身体

## 配置说明

### 位置配置
角色位置会自动保存到 `data/config.json`

### 手动修改配置
编辑 `data/config.json`:
```json
{
    "window_x": 100,
    "window_y": 100,
    "character_type": "live2d"
}
```

## 替换角色图片

### 方法 1：使用 Live2D 模型
1. 下载 Live2D 免费模型：https://www.live2d.com/en/download/sample-data/
2. 解压到 `assets/live2d/` 目录
3. 修改 `character_widget.py` 中的纹理路径

### 方法 2：使用自定义图片
1. 准备 PNG 图片（透明背景，300x500px）
2. 放到 `assets/characters/` 目录
3. 在 `character_widget.py` 中加载：
```python
self.load_texture("assets/characters/your_character.png")
```

## 常见问题

### Q: 角色显示为方框/emoji？
A: 检查纹理图片路径是否正确，或图片文件是否存在。

### Q: 拖动后位置没有保存？
A: 确保 `data/config.json` 文件有写入权限。

### Q: 聊天窗口打不开？
A: 检查是否有多个实例运行，关闭所有实例后重试。

### Q: Live2D 模型不显示？
A: 需要加载正确的纹理路径，检查 `character_widget.py` 中的路径配置。
