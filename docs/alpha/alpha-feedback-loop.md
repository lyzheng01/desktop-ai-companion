# Alpha Feedback Loop

## Welcome Message

欢迎来测这个住在桌面上的 AI 小伙伴。

这个版本还很早期，我最想知道的是：她有没有真的让你觉得“在桌面上陪着你”，而不是只是一个聊天框。

如果你愿意，尽量多告诉我这些：

- 你有没有真的把她留在桌面上
- 哪个瞬间让你觉得她“活了”
- 哪个地方最烦、最出戏、最像工具

## Daily Questions

- 今天有没有打开她？
- 你今天有没有主动点她聊天？
- 哪个瞬间让你觉得她“活了”？
- 哪个地方最烦？

## Weekly Questions

- 你更把她当工具还是伙伴？
- 如果只能保留一个功能，你会保留什么？
- 你明天还会继续开着她吗？
- 她更像“陪伴你”，还是“只是能回答问题”？

## Bug Report Format

- 发生了什么
- 你当时在做什么
- 是否稳定复现
- 你原本期待什么
