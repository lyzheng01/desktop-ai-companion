# P0 Demo Scripts

## Script A: Presence
1. 角色待在桌面角落
2. 用户点击角色
3. 输入“今天有点累”
4. 角色进入 thinking 状态
5. 回复一句温柔的话
6. 回到安静待机

## Script B: Utility
1. 用户正在写文档或代码
2. 卡住了
3. 点击角色提问
4. 角色 thinking
5. 给出简洁帮助
6. 用户继续工作，角色安静待着

## Script C: Reaction
1. 待机
2. listening
3. thinking
4. talking
5. hide / quiet presence
