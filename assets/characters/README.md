# 角色资源目录

## 二次元风格角色

### 小樱 (Sakura)
- **性格**: 温柔治愈
- **外观**: 粉色短发，白色连衣裙，樱花发饰
- **动作**: 眨眼、歪头、双手托腮、开心摇摆

### 阿白 (Shiro)
- **性格**: 理性知性
- **外观**: 白发少年，戴眼镜，穿灰色卫衣
- **动作**: 推眼镜、思考托下巴、挥手、打哈欠

### 奈奈 (Nana)
- **性格**: 元气活泼
- **外观**: 双马尾猫耳少女，黑色 + 橙色配色
- **动作**: 猫耳抖动、舔爪子、伸懒腰、玩毛线球

## 图片格式规范

- **格式**: PNG (支持透明背景)
- **尺寸**: 200x300px (角色站立)
- **帧率**: 8-12fps (简单动画)
- **命名**: `{角色名}_{动作名}_{帧号}.png`

示例：
```
sakura_idle_01.png
sakura_idle_02.png
sakura_blink_01.png
shiro_think_01.png
nana_ear_twitch_01.png
```
